google-play-store-info
======================

Google Play Storeのアプリ情報をスクレイピングしてJSONフォーマットで返すよ

http://{server_path}/app-detail?id={app_package_name}

```
{
  "message":"app fetched",
  "data":{
    "package":"jp.tell_me.android",
    "name":"テルミー 悩み・相談から質問まで99％回答掲示板アプリ",
    "version":"2.0.5"
  }
}
```
